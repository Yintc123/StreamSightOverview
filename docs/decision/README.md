# Decision Records

架構與技術決策紀錄(ADR）。每份文件記錄一個決策的**背景、選擇、取捨,以及被否決的替代方案**,
讓後人(和未來的自己)知道「為什麼當初這樣決定」。

## 慣例

- 檔名:`NNNN-kebab-case-title.md`,編號遞增。

- 決策一旦被取代,不刪除舊檔,改標記 `Superseded` 並連到新的。
